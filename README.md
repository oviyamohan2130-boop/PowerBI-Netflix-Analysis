# Netflix Content Analysis Dashboard

## Project Overview

This project is a Power BI dashboard created to analyze Netflix content data. The dashboard provides insights into movies and TV shows available on Netflix, their release trends, genres, and ratings.

## Tools Used

* Power BI
* Excel

## Dashboard Features

* Distribution of Movies and TV Shows
* Netflix Content Release Trends
* Top 10 Genres Available on Netflix
* Content Rating Analysis

## Key Insights

* Movies make up the majority of Netflix content.
* Content releases increased rapidly after 2010.
* Drama and International Movies are among the most popular genres.
* TV-MA is the most common content rating.
